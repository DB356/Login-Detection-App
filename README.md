# 🔐 Login Detection App

A Streamlit-based web application that detects suspicious login attempts using parameters like IP address, login time, and number of attempts.

---

## 🚀 Features
- Detects suspicious login activity based on:
  - IP address  
  - Login time  
  - Number of login attempts  
- Real-time detection  
- Simple and interactive UI  

---

## 🧠 How It Works
The app analyzes:
- IP address  
- Login time patterns  
- Number of attempts  

Based on predefined logic or a machine learning model, it classifies login attempts as:
- Normal  
- Suspicious  

---

## 🛠️ Tech Stack
- Python  
- Streamlit  
- Pandas  
- NumPy  
- Scikit-learn  

---

## 📦 Installation

git clone https://github.com/DB356/login-detection-app.git  
cd login-detection-app  
pip install -r requirements.txt  
streamlit run app.py
